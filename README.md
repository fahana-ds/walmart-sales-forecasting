# Walmart Sales Forecasting

This project predicts Walmart weekly sales using Machine Learning and Time Series Forecasting models.

## Models Used
- Linear Regression
- Random Forest Regressor
- Prophet Forecasting

## Techniques
- Feature Engineering
- Lag Features
- Rolling Mean
- Store-wise Forecasting
- MAE & RMSE Evaluation

## Final Result
Prophet and Random Forest models were compared store-wise to forecast future weekly sales trends.
